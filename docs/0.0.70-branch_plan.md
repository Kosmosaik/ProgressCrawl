
# 0.0.70 – Exploration (Zones)  
## Full Branch & Sub‑Branch Plan (Recommended Structure)

This document contains the finalized list of branches/sub-branches for Update **0.0.70 – Exploration**, separated according to implementation complexity and recommended development workflow.

---

# **0.0.70a — Zone Core System (Skeleton)**  
### *Goal:* Basic Zone structure, tile revealing, and exploration loop.

**Includes:**
- Zone object/class with essential fields  
- Tilemap structure (walkable, blocked, locked)  
- Exploration system (2–5s ticks, tile reveal, % completion)  
- Basic discovery list (generic entries)  
- Debug button: “Enter Starting Zone”  
- Finish Zone menu: **[STAY] / [LEAVE ZONE]**

---

# **0.0.70b — World Map & Zone Generation**
### *Goal:* Global map, Fog of War, and transitions.

**Includes:**
- World Map grid  
- Fog of War states  
- Starting Zone placement  
- Adjacent zone generation  
- Pre-generation of:
  - Tilemap shapes  
  - Blocked regions  
  - Locked regions  
  - Default weather state  
- Transitions:
  - Zone → World Map  
  - World Map → Zone  

---

# **0.0.70c — Static Zone Content Generation**
### *Goal:* Pre-generate all static Zone content before entering.

This branch is internally worked in steps, but one version in patch notes.

**Includes:**
- Biome definitions (first pass)
- Static pre-generation of:
  - Resource Nodes  
  - Static Entities  
  - Locations  
  - POIs  
  - Locked region contents  
- Tile reveal shows pre-generated content

**Internal implementation steps (not separate branches):**
1. Resources  
2. Entities  
3. Locations + POIs  

---

Split into 3 sub-branches:

---

- Roaming enemies

- Encounter UI feedback

---

## **0.0.70d3 — World Flavor Events**
- Disasters (storms, quakes, fires)  
- Travelling merchants  
- Wanderers  
- Seasonal event triggers  
- Scripted Zone events  

---

# **0.0.70e — Resource Interaction Layer**
### *Goal:* Interact with resource nodes inside Zones.

**Sub-steps inside this branch:**

## **0.0.70e1 — Basic Interaction**
- “Inspect” window  
- Node info display  

## **0.0.70e2 — Harvesting + Depletion**
- Harvest action  
- Depleted node state  
- Removing from discovery list or marking “Depleted”

## **0.0.70e3 — Respawn System**
- Individual respawn timers (2–4 in-game days)  
- Reroll grade, subtype, and placement on respawn
